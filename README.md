<h1>Quick Start Guide</h1>
<p>git clone https://github.com/Shoblik/lightspeedvoice-sample.git</p>
<p>composer install</p>
<p>Copy .env.example into a new file called .env</p>
<p>run php artisan key:generate</p>
<p>create test schema 'laravel' in your DB</p>
<p>php artisan migrate:fresh</p>
<p>php artisan db:seed</p>
